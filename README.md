# Pokimon-Type-Battle-Game
Built a Mini Pokémon Type Battle Game in Java!


A beginner-friendly Java console game inspired by Pokémon type battles. The game uses Fire, Water, and Grass types, where the player competes against the computer using randomly generated choices.

🚀 Features
🔥 Fire, 💧 Water, and 🌿 Grass battle system
🎲 Random computer opponent
🏆 Win, Lose, or Draw results
📚 Built using Java fundamentals
💻 Console-based gameplay
🛠️ Technologies Used
Java
Scanner Class
Random Class
Conditional Statements (if-else)
🎮 Type Matchups
🔥 Fire beats 🌿 Grass
💧 Water beats 🔥 Fire
🌿 Grass beats 💧 Water
